## File Naming Convention

```
YYYY-MM-DD-slug-[lang].md
```

| Suffix | Language |
|--------|----------|
| `-fr.md` | French (original) |
| `-en.md` | English (translation) |

---

## Frontmatter Fields

| Field | Description |
|-------|-------------|
| `title` | Full post title |
| `slug` | URL-safe identifier, shared across both language versions |
| `lang` | ISO 639-1 language code (`fr` or `en`) |
| `date` | Publication date (YYYY-MM-DD) |
| `author` | Byline |
| `tags` | Array of topic tags |
| `canonical_url` | Intended permalink on ayitiintel.com |
| `paired_translation` | Filename of the corresponding language version |
