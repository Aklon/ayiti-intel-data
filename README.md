# AYITI INTEL — Data Repository

Structured historical datasets powering the [AYITI INTEL](https://ayitiintel.com) intelligence platform — built for NGOs, embassies, and research institutions studying Haitian history and political economy.

---

## Datasets

| Dataset | Events | Description |
|---------|--------|-------------|
| [`haiti-historical-patterns/`](./haiti-historical-patterns/) | 46 | Four structural dependency cycles: 1825 French debt · 1915–34 US occupation · 1991–94 Cédras junta · 2004 coup |

---

## Format

All datasets are published as structured JSON with:
- Embedded field definitions (`field_definitions` key)
- Controlled vocabularies documented in each dataset's `README.md`
- Human-readable labels and machine-sortable rank fields
- Dataset-level stats (counts by theme, type, impact)
